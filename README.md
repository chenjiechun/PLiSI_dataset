# PLiSI_dataset
## Brief description
This dataset can be used to evaluate the methods for localizing pupils in a pair of stereo images. The dataset consists of two sub-datasets: a synthetic sub-dataset and a real-world sub-dataset. The real-world sub-dataset contains 17 NIR video files. The video files were captured by using a self-developed and stereo vision based pupilometer. The synthetic sub-dataset consists of 9,124 pairs of stereo edge maps, and the edge maps are further divided into three sub-datasets that simulate three types of pupil contour defects respectively.

PLiSI-s1 sub-dataset: This sub-dataset consists of 2,670 pairs of stereo edge maps. The edge maps contain synthetic pupil contours used to simulate the defective pupil contours that are attached with non-pupil contours. This sub-dataset is further divided into two sub-datasets: PLiSI-s11 and PLiSI-s12. In PLiSI-s11 sub-dataset, each pair of edge maps contain one defective pupil contour, whereas in PLiSI-s12 sub-dataset, there are teo defective pupil contours in each pair of edge maps.

PLiSI-s2 sub-dataset: This sub-dataset contains 4,054 pairs of stereo edge maps in total, and is further divided into two sub-datasets: PLiSI-s21 and PLiSI-s22. The edge maps contain synthetic pupil contours used to simulate the defective pupil contours that are split into multiple segments. In PLiSI-s21 sub-dataset, each pair of edge maps contain one defective pupil contour, whereas in PLiSI-s22 sub-dataset, there are teo defective pupil contours in each pair of edge maps.

PLiSI-s3 sub-dataset: This sub-dataset consists of 2,400 pairs of stereo edge maps. The edge maps contain synthetic pupil contours used to simulate the defective pupil contours that are partially occluded by eyelids. This sub-dataset is further divided into four sub-datasets: PLiSI-s31, PLiSI-s32, PLiSI-s33, and PLiSI-s34. These sub-datasets differ in the range of the central angle of the pupil contours that can be used to determine pupil locations, with the ranges of π/2 - 2π/3, 2π/3 - 5π/6, 5π/6 – π, and π - 7π/6 respectively.
